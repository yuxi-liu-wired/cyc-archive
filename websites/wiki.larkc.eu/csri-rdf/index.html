Computer Scientists' Retained Interests Dataset

Introduction about the Dataset

User interests are very relevant to queries that users input. They are changing dynamically. The dynamic user Interests are acquired and can be partially predicted. We analized all the computer scientists' publication (currently including 615124 persons) in the DBLP dataset (Currently, we use the SwetoDBLP version of the dataset) and we made a prediction of all their future interests in the year 2009 and created an RDF file containing top 9 interests of all the authors. Download the interest RDF file (updated version: 2010-Jan-17th).

Maintenance of the Dataset

This dataset is released under the LarKC project Work Package 2 and Work Package 4 by International WIC Institute (WICI), and will be updated every 4 month.

Illustrative Examples to use the Dataset

In scientific literature search, vague or incomplete queries can be refined using implicitly or explicitly acquired user interests. These dataset can be used to refine the query from different researchers when they want to search literatures on DBLP.
